Culturify

Culturify was developed over 2 weeks with four other La Fosse Academy trainees. It is a culinary social media app aiming to bring people from different cultures together through the medium of food.

My contribution:
I spent most of my time developing the Flask backend. This consisted of creating the numerous routes for the application, writing automated tests, and structuring the database.
In the React frontend, I ensured that data was received from the backend for various parts of the application. I also created the forms to post new recipes and share them as posts.

Technologies:
Python, Flask, JavaScript, React, Vite, PostgreSQL, Docker, Pytest, Vitest.

Pomodogo

Pomodogo was developed over the course of a week with three other La Fosse Academy trainees. It uses a gamified pomodoro timer to help users stay focused and complete tasks. It also has cute animations which calms users and hopefully brings them joy.

My contribution:
I mainly worked on the frontend as I wanted to improve my React and CSS skills. As a result, I developed the functionality for the Pomodoro timer, which included getting the animations to behave exactly as we wanted in each scenario.

Technologies:
React, Vite, JavaScript, MongoDB, Jest, Vitest.

Geo Quiz

Geo Quiz was developed in a week as part of a 4-person team. This educational site has 3 quizzes which should help users to learn more about the countries of the world in a fun and interactive way. The leaderboard section gives users the incentive to keep learning so that they can better their score. There is also a flashcard section in which users can learn about random countries without any time contraints.

My contribution:
I predominantly worked on the backend and the implementation of all the game features. I also added the functionality to the leaderboard.

Technologies:
Node, JavaScript, HTML, CSS, Express.

Solo Projects

Shape Invaders

Shape Invaders was made during a game jam in 2023. The theme was "life in 2 dimensions". The aim of the of the game is to destroy as many of the hostile shapes as possible. You need to use the blue bullet to shoot the blue enemies, and the red bullet to shoot the red ones. The game gets progressively more difficult, the longer you survive.

Technologies: C#, Unity.
